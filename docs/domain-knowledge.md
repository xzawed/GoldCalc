## 무게 단위 변환

| 단위 | 그램(g) 환산 | 비고 |
|------|------------|------|
| 1 troy oz | 31.1035 g | 국제 금시세 기준 단위 |
| 1 돈 | 3.75 g | 한국 소매 금 거래 기준 |
| 1 냥 | 37.5 g | = 10돈 |
| 1 관 | 3,750 g | = 1,000돈 (UI 미제공) |

**소스:** `src/utils/metalCalc.ts` — `DON_TO_G = 3.75`, `TROY_OZ_TO_G = 31.1035`

## 계산 공식

```
금 1g 원화 = (XAU/USD ÷ 31.1035) × USD/KRW × 순도비율
금 1돈 원화 = 금 1g 원화 × 3.75

은 1g 원화 = (XAG/USD ÷ 31.1035) × USD/KRW × 순도비율

국내금 원화 = KRX 고시가(원/g) × 순도비율   ← 환율 변환 불필요
국내은 원화 = 은 1g 원화 (국제시세 환산)     ← KRX 은 현물시장 없음
```

**모든 원화 표시:** `Math.round()` 원 단위 반올림  
**천 단위 포맷:** `toLocaleString('ko-KR')` → `₩1,234,567`

## 국내금 vs 국제금 차이

| 구분 | 국제금 (XAU) | 국내금 (KRX) |
|------|------------|------------|
| 기준 | LBMA 런던 금시장 USD/troy oz | KRX 한국거래소 원/g |
| API | GoldAPI.io → `/api/gold-price` | data.go.kr → `/api/domestic-gold` |
| 환율 필요 | 예 (USD/KRW) | 아니오 |
| 히스토리 | 날짜별 GoldAPI.io | data.go.kr 기간 조회 |
| 예측(3구역) | 제공 | 미제공 |

# PRD — GoldCalc 귀금속 시세 계산기

**버전**: 1.1
**작성일**: 2026-03-18
**상태**: 초안 (Draft)

---

## 1. 배경 및 목적

### 문제 정의
국제 금 거래는 트로이 온스(troy oz) 단위의 USD 가격을 기준으로 하지만,
한국 소비자와 소매 금 시장은 **돈(3.75g)** 단위와 **원화(KRW)** 를 사용한다.
은(Silver) 역시 국제 시세(XAG/USD)를 원화로 환산해야 하며, 국내 금시장(KRX)의 원화 직접 시세도 별도 확인이 필요하다.
소비자가 귀금속을 구매하거나 보유 자산 가치를 확인하려면 여러 단계의 환산을 직접 계산해야 하는 불편이 있다.

### 목적
국제 금시세 및 은시세를 실시간으로 조회하여 한국식 단위(g, 돈, 냥)와 원화로 즉시 환산하고,
국내 금시세(KRX 금시장) 원화 직접 조회를 지원하며,
과거 시세 변동 내역과 단기 가격 예측을 함께 제공하는 웹 서비스를 구축한다.

### 목표 사용자
- 금 실물(골드바, 금반지 등)을 구매 또는 판매하려는 일반 소비자
- 금 자산 가치를 주기적으로 확인하는 개인 투자자
- 금 시세 동향을 파악하려는 관심자

## 2. 범위

### In Scope (v1.1)
- 국제 금시세(XAU/USD) 실시간 조회
- 국제 은시세(XAG/USD) 실시간 조회 및 원화 환산
- 국내 금시세(KRX 금시장) 원화 직접 조회 및 계산
- 국내 은시세(국제 XAG/USD → KRW 환산) 원화 계산
- USD/KRW 환율 실시간 조회
- 자산 탭 네비게이션 (국제금 / 국제은 / 국내금 / 국내은)
- g / 돈 / 냥 단위 입력 → 원화 환산 계산기
- 금 순도(24K / 18K / 14K) 선택
- 은 순도(999 / 925 / 900 / 800) 선택
- 날짜별 시세 변동 히스토리 차트 및 테이블 (최대 1년)
- 이동평균 + 선형 회귀 기반 단기 가격 예측 (7일 / 30일)
- 반응형 웹 UI (모바일 / 데스크톱)

### Out of Scope (v1.1 제외)
- 회원가입 / 로그인 / 마이페이지
- 알림(push/email) 기능
- 백금 등 기타 귀금속 지원
- 실제 거래 및 결제 기능
- 네이티브 앱 (iOS / Android)

## 4. 기능 요구사항

### 4.1 [1구역] 실시간 계산기

#### FR-01 실시간 금시세 조회
- GoldAPI.io에서 XAU/USD 현재 가격을 fetch
- ExchangeRate-API에서 USD/KRW 환율을 fetch
- 앱 진입 시 즉시 조회, 이후 5분 간격 자동 갱신
- 마지막 업데이트 시각을 "2026-03-18 14:32 기준" 형태로 표시

#### FR-02 무게 입력 및 단위 변환
- 숫자 입력 필드 1개 (소수점 허용)
- 단위 탭: **g** / **돈** / **냥** 중 선택
- 단위 변환 상수: 1돈 = 3.75g, 1냥 = 37.5g
- 단위 변경 시 입력값 유지하며 결과만 재계산

#### FR-03 순도 선택
- 선택지: 24K (99.99%) / 18K (75%) / 14K (58.3%)
- 기본값: 24K

#### FR-04 원화 환산 결과 표시
- 계산 결과를 원 단위 반올림(Math.round)하여 표시
- 천 단위 콤마 포맷: `toLocaleString('ko-KR')`
- 표시 예: `3돈 × 24K = 1,234,567원`
- 기준 단가(원/g, 원/돈)도 함께 표시

#### FR-14 국제 은시세 계산기
- GoldAPI.io에서 XAG/USD 현재 가격을 fetch
- 은 순도 선택: 999 (99.9%) / 925 (92.5%) / 900 (90.0%) / 800 (80.0%)
- 기본값: 999
- 계산 공식: `은 1g 원화 = (XAG/USD ÷ 31.1035) × USD/KRW × 순도비율`
- 단위 입력 및 결과 표시는 금 계산기(FR-02, FR-04)와 동일 UX 적용

#### FR-15 국내 금시세 계산기
- data.go.kr(KRX 금시장) API로 국내 금 원화 시세를 직접 조회
- Railway Express 서버 프록시 경유로 CORS 우회
- 원화 기준 시세이므로 환율 변환 불필요
- 순도 선택(24K/18K/14K) 및 단위 입력은 국제 금 계산기와 동일

#### FR-16 자산 탭 네비게이션
- 계산기 상단에 **국제금** / **국제은** / **국내금** / **국내은** 탭 배치
- 탭 전환 시 해당 자산의 계산기·시세·순도 옵션으로 전환
- 기본 선택: 국제금

#### FR-17 국내 은시세 계산기
- 국제 XAG/USD 시세를 실시간 USD/KRW 환율로 환산하여 KRW/g 표시
- KRX에 은 현물 시장이 없으므로 국제 가격 환산 방식 채택
- 은 순도 선택: 999 / 925 / 900 / 800
- 기본값: 999
- 히스토리 차트·테이블·최고/최저/평균 배지 제공
- 예측(3구역) 미제공 (국내 금과 동일 정책)

### 4.2 [2구역] 시세 변동 내역

#### FR-05 기간 탭 선택
- 탭: **1주** / **1개월** / **3개월** / **1년**
- 탭 전환 시 차트와 테이블 동시 갱신
- 기본값: 1개월

#### FR-06 시세 변동 차트
- Recharts `ComposedChart` 사용
- X축: 날짜, Y축 좌: USD/oz, Y축 우: 원화/g (이중 Y축)
- 반응형 너비 (`ResponsiveContainer`)
- 커서 호버 시 툴팁으로 날짜별 상세값 표시

#### FR-07 요약 배지
- 선택 기간 기준 **최고가** / **최저가** / **평균가** 배지 3개 표시
- 각 배지에 USD/oz 및 원화/g 동시 표시

#### FR-08 날짜별 시세 테이블
- 컬럼: 날짜 / 국제금시세(USD/oz) / 환율(USD/KRW) / 원화/g / 전일대비
- 전일대비 등락 색상: 상승 빨강(`#ef4444`), 하락 파랑(`#3b82f6`), 보합 회색
- 최신 날짜가 상단 (내림차순 정렬)
- 스크롤 또는 페이지네이션으로 전체 조회 가능

### 4.3 [3구역] 금시세 예측

#### FR-09 예측 기간 선택
- 선택지: **7일** / **30일**
- 기본값: 7일

#### FR-10 예측 차트
- 과거 실제값(실선)과 예측값(점선)을 하나의 차트로 연결
- 신뢰 구간(±1σ)을 반투명 영역(`fillOpacity: 0.15`)으로 표시
- 과거/예측 경계선을 수직 점선 또는 배경색으로 시각적 구분

#### FR-11 예측 알고리즘
- MA5 / MA20 이동평균 교차로 단기 트렌드 방향 판단
- 최근 90일 데이터 기반 선형 회귀로 예측값 산출
- 신뢰 구간: 과거 90일 실제값의 표준편차(σ) 기반 ±1σ

#### FR-12 시장 신호 요약
- 예측 섹션 하단에 텍스트 카드로 표시
- 항목: 달러 인덱스(DXY) 방향 / 미국 국채 10년물 금리 / VIX 지수
- 각 항목의 현재값과 최근 추세(↑/↓) 표시

#### FR-13 면책 문구 (필수)
- 예측 섹션 최하단에 항상 표시
- 문구: `"본 예측은 통계적 추세 분석에 기반한 참고 정보이며, 투자 조언이 아닙니다. 실제 금 가격은 다양한 요인에 의해 예측과 다를 수 있습니다."`
- 숨김 처리 또는 조건부 렌더링 금지

## 5. 비기능 요구사항

### 5.1 성능
| 항목 | 목표 |
|------|------|
| 최초 페이지 로드 | 3초 이내 (LCP 기준) |
| 계산기 결과 반영 | 입력 즉시 (debounce 없음) |
| API 응답 대기 시간 | 2초 초과 시 로딩 스피너 표시 |
| 히스토리 차트 렌더링 | 데이터 수신 후 500ms 이내 |

### 5.2 가용성 및 에러 처리
- API 호출 실패 시 마지막 정상 캐시값을 유지하고 에러 배너 표시
- 네트워크 오프라인 감지 시 "시세를 불러올 수 없습니다" 메시지 표시
- 모든 API 호출에 타임아웃 10초 적용

### 5.3 반응형 디자인
| 브레이크포인트 | 레이아웃 |
|--------------|---------|
| 모바일 (< 768px) | 단일 컬럼, 차트 높이 220px |
| 태블릿 (768px~) | 단일 컬럼, 차트 높이 280px |
| 데스크톱 (1024px~) | 최대 너비 1200px 중앙 정렬, 차트 높이 350px |

### 5.4 보안
- API 키는 환경변수(`.env`)로 관리, 소스코드에 하드코딩 금지
- `.env` 파일은 `.gitignore`에 반드시 포함
- 외부 API 호출은 HTTPS만 사용

### 5.5 접근성
- 색상만으로 정보를 전달하지 않음 (등락: 색상 + 화살표 아이콘 + 부호 병행)
- 주요 수치 요소에 `aria-label` 적용
- 키보드 탐색 가능한 탭/버튼 구성

## 7. 데이터 요구사항

### 7.1 외부 데이터 소스

| 데이터 | 출처 | 갱신 주기 | 비고 |
|--------|------|----------|------|
| 국제 금시세 (현재) | GoldAPI.io | 앱 기준 5분 | API 키 필요 |
| 국제 금시세 (히스토리) | GoldAPI.io | 일 1회 | 날짜별 엔드포인트 |
| USD/KRW 환율 | ExchangeRate-API | 앱 기준 5분 | 무료 플랜 가능 |
| 달러 인덱스 (DXY) | Alpha Vantage | 일 1회 | 무료 플랜 일 25회 한도 |
| 미국 국채 10년물 | FRED API | 일 1회 | 무료 |
| VIX 지수 | Alpha Vantage | 일 1회 | DXY와 동일 API |
| 국제 은시세 (현재) | gold-api.com | 앱 기준 5분 | 인증 불필요 |
| 국내 금시세 | data.go.kr | 앱 기준 5분 | Railway 프록시 경유 |
| 국내 은시세 (현재+히스토리) | gold-api.com (XAG) + ExchangeRate-API | 앱 기준 5분 | 별도 API 불필요, 환율 환산 |

### 7.2 캐싱 전략
- 현재 금시세/환율: 5분 캐시 (메모리)
- 히스토리 데이터: 24시간 캐시 (localStorage)
- 예측 결과: 히스토리 데이터 변경 시에만 재계산

### 7.3 무료 플랜 운영(과금 방지) 정책
- 본 프로젝트는 모든 외부 API를 **무료 플랜 기준**으로만 운영한다는 전제를 가진다.
- 보조지표(Alpha Vantage: DXY/VIX, FRED: DGS10)는 **추가 과금 없이** 운영하기 위해, 문서에 정의된 `staleTime: 24시간`(하루 1회 갱신)과 캐시 활용을 반드시 준수한다.
- 특히 Alpha Vantage 무료 플랜은 요청 한도가 존재하므로, 페이지 렌더/탭 전환/컴포넌트 리렌더로 인해 동일 요청이 반복되지 않도록(쿼리 캐시/stale 정책 준수) 구현한다.

컴퓨터에 익숙하지 않은 분도 따라 하실 수 있도록 단계별로 설명합니다.

## 1. 은시세 API — 확인만 하면 됩니다

은시세는 **gold-api.com** 이라는 무료 서비스를 사용합니다.
이 서비스는 **별도 가입이나 키 발급 없이** 바로 사용할 수 있습니다.

아래 확인 작업만 해주세요:

### 확인 방법

1. 인터넷 브라우저(크롬, 엣지 등)를 엽니다
2. 주소창에 아래 주소를 복사해서 붙여넣고 Enter를 누릅니다:
   ```
   https://api.gold-api.com/price/XAG
   ```
3. 화면에 아래와 비슷한 내용이 나오면 정상입니다:
   ```
   {"price": 30.25, "currency": "USD", "metal": "XAG", ...}
   ```
   - `price` 옆의 숫자가 현재 은 1온스의 달러 가격입니다
   - 이런 내용이 보이면 은시세 API는 정상 작동하는 것입니다

4. 만약 오류 메시지가 나오거나 빈 화면이 보이면, 이 내용을 알려주세요

> **이 단계에서 회원가입이나 키 발급은 필요 없습니다.**

## 2. 국내 금시세 API — 공공데이터포털 키 발급

국내 금시세는 대한민국 정부가 운영하는 **공공데이터포털(data.go.kr)** 에서 제공합니다.
무료로 사용할 수 있지만, 회원가입과 API 키 발급이 필요합니다.

### 2-1단계: 공공데이터포털 회원가입

1. 브라우저에서 아래 주소로 이동합니다:
   ```
   https://www.data.go.kr
   ```

2. 화면 오른쪽 상단의 **"회원가입"** 버튼을 클릭합니다

3. 가입 유형을 선택합니다:
   - **"일반 회원"** 을 선택하세요 (개인 사용자용)

4. 본인인증을 진행합니다:
   - 휴대폰 인증 또는 아이핀 인증 중 편한 방법을 선택합니다
   - 화면에 나오는 안내에 따라 인증을 완료합니다

5. 가입 정보를 입력합니다:
   - 아이디, 비밀번호, 이름, 이메일 등을 입력합니다
   - 모두 입력한 후 **"가입하기"** 버튼을 클릭합니다

6. 가입이 완료되면 **로그인** 합니다

### 2-2단계: 금시세 API 활용 신청

1. 로그인한 상태에서 아래 주소로 이동합니다:
   ```
   https://www.data.go.kr/data/15094805/openapi.do
   ```
   - 이 페이지는 "금융위원회_일반상품시세정보" API 페이지입니다

2. 페이지 중간에 있는 **"활용신청"** 버튼을 클릭합니다
   - 버튼이 보이지 않으면 페이지를 아래로 스크롤해 보세요

3. 활용 목적을 입력하는 화면이 나옵니다:
   - **활용 목적** 란에 아래와 같이 입력합니다:
     ```
     개인 프로젝트에서 국내 금시세 정보를 조회하기 위해 사용합니다
     ```
   - **동의** 체크박스가 있으면 모두 체크합니다
   - **"신청"** 버튼을 클릭합니다

4. 신청이 완료되면 **자동으로 API 키(인증키)가 발급**됩니다
   - 보통 즉시 발급되지만, 경우에 따라 몇 시간이 걸릴 수 있습니다

### 2-3단계: 발급된 API 키 확인하기

1. 공공데이터포털 상단 메뉴에서 **"마이페이지"** 를 클릭합니다

2. 왼쪽 메뉴에서 **"오픈API"** → **"인증키 발급현황"** 을 클릭합니다

3. 목록에서 "금융위원회_일반상품시세정보"를 찾습니다

4. **"일반 인증키(Encoding)"** 라고 적힌 긴 문자열을 찾습니다
   - 이것이 API 키입니다
   - 예시 형태: `Abc123def456ghi789jkl012mno345pqr678stu901vwx234==`
   - 매우 긴 문자열입니다 (보통 80~100자)

5. 이 키를 **복사**합니다:
   - 키 옆에 "복사" 버튼이 있으면 클릭합니다
   - 없으면 키 전체를 마우스로 드래그해서 선택한 후 `Ctrl + C`로 복사합니다

> **중요**: 이 키는 비밀번호처럼 취급하세요. 다른 사람에게 공유하지 마세요.

## 3. 발급받은 키를 프로젝트에 등록하기

프로젝트 폴더에 있는 `.env` 파일에 키를 추가해야 합니다.

### 방법

1. 프로젝트 폴더(`D:\Source\GoldCalc`)를 엽니다

2. `.env` 파일을 찾습니다
   - 이 파일은 숨김 파일일 수 있습니다
   - 파일이 보이지 않으면 Windows 탐색기 상단의 "보기" → "숨긴 항목"을 체크하세요
   - `.env` 파일이 아예 없으면, `.env.example` 파일을 복사해서 이름을 `.env`로 변경하세요

3. `.env` 파일을 메모장 또는 텍스트 편집기로 엽니다:
   - 파일을 마우스 오른쪽 클릭 → "연결 프로그램" → "메모장" 선택

4. 파일 맨 아래에 아래 줄을 추가합니다:
   ```
   DATA_GO_KR_API_KEY=여기에_복사한_키를_붙여넣으세요
   ```
   - `여기에_복사한_키를_붙여넣으세요` 부분을 지우고, 2-3단계에서 복사한 키를 `Ctrl + V`로 붙여넣습니다
   - 키 앞뒤에 따옴표(`"`)나 공백을 넣지 마세요
   - 올바른 예시:
     ```
     DATA_GO_KR_API_KEY=Abc123def456ghi789jkl012mno345pqr678stu901vwx234==
     ```

5. 파일을 저장합니다 (`Ctrl + S`)

## 4. Vercel(배포 서버)에 키 등록하기

웹사이트를 인터넷에 공개(배포)하려면, Vercel에도 키를 등록해야 합니다.
로컬(내 컴퓨터)에서만 테스트할 거라면 이 단계는 나중에 해도 됩니다.

### 방법

1. 브라우저에서 Vercel 대시보드로 이동합니다:
   ```
   https://vercel.com/dashboard
   ```

2. 로그인 후, GoldCalc 프로젝트를 클릭합니다

3. 상단 탭에서 **"Settings"** 를 클릭합니다

4. 왼쪽 메뉴에서 **"Environment Variables"** 를 클릭합니다

5. 새 환경변수를 추가합니다:
   - **Key** 란에: `DATA_GO_KR_API_KEY`
   - **Value** 란에: 공공데이터포털에서 발급받은 키를 붙여넣습니다
   - **Environment** 체크박스: `Production`, `Preview`, `Development` 모두 체크
   - **"Save"** 버튼을 클릭합니다

6. 환경변수가 목록에 추가된 것을 확인합니다

7. 변경사항을 적용하려면 **재배포**가 필요합니다:
   - 상단 탭에서 "Deployments" 클릭
   - 가장 최근 배포 항목의 오른쪽 `⋮` (점 세 개) 클릭
   - **"Redeploy"** 클릭
